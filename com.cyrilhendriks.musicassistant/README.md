# Music Assistant for Homey Pro

Homey Pro app that adds support for Music Assistant via its JSON API.

## Features (MVP)
- Play a radio station on a speaker or group
- Play a playlist on a speaker or group
- Play a track (search) on a speaker or group
- Announcements to selected speakers with best-effort resume

## Development
### Requirements
- Node.js (recommended: 20)
- Homey CLI

### Install
```bash
npm install
